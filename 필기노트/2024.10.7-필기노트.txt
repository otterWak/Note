20241007(월) DB

====================
[ DB 개요 ]

1. Data와 Database
1) Data
- 관찰 결과로 나타난 정량적 혹은 정성적인 실제 값
2) 정보
- 데이터를 기반으로 의미를 부여하는 것
3) Database
- 한 조직에 필요한 정보를 여러 응용 시스템에서 공용할 수 있도록
논리적으로 연관된 데이터를 모으고 중복되는 데이터를 최소화하여
구조적으로 통합/저장해놓은 것
- 무결성을 지킨다는 특징이 있음

2. DBMS(Database Management System)
- 데이터베이스에서 데이터 추출, 조작, 정의, 제어 등을 할 수 있게 해주는
데이터베이스 전용 관리 프로그램
- 기능: 데이터 추출/조작/정의(구조)/제어
- 종류 : Access/SQL Server/Oracle/MySQL/MriaDB/SQLite...
- 사용 이점
1) 데이터 독립화
2) 데이터 중복 최소화, 데이터 무결성 보장
3) 데이터 보안 향상
4) 관리 편의성 향상
